# Tishman Speyer (tishman-speyer)

Tishman Speyer is a privately held, global real estate developer, operator, and investment manager founded in 1978 and headquartered at 45 Rockefeller Plaza in New York City. The firm acquires, develops, operates, and manages first-class office, residential, retail, industrial, mixed-use, and life-science properties across roughly 40 markets in the Americas, Europe, and Asia. Its signature portfolio includes Rockefeller Center, the MetLife Building, the MesseTurm in Frankfurt, The Spiral at Hudson Yards, and Mission Rock in San Francisco. The firm has reported approximately $65 billion in assets under management. This repository captures the public surface, developer footprint, and machine-readable API artifacts for Tishman Speyer.

## APIs

Tishman Speyer does not publish a public developer portal, OpenAPI specifications, SDKs, CLIs, or open-source repositories. The firm operates a GitHub organization at `github.com/tishmanspeyer` that has no public members and no public repositories. There are no documented API artifacts to catalog at this time.

Adjacent digital surfaces — the **Zo** tenant-experience mobile app, the **Studio** flex-workplace platform, the **Breakthrough Properties** life-science joint venture with Bellco Capital, and the **TS Ventures** proptech investment arm — are end-user / corporate properties without published partner or developer APIs.
